git remote add origin git@github.com:dengbowang/text.git
git branch -M main

git远程覆盖本地。。

master 标红是远程分支名 记得替换下 一般都是在dev 或者版本号上开发

git 强制覆盖本地/远程仓库

强制覆盖远程仓库 :

`git push -f origin master`

强制覆盖本地仓库(假设你的分支是master):

git gc

git reset

git reset --hard master
